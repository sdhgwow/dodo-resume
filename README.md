# Dodo Resume

Интерактивное резюме-портфолио в стиле Dodo, созданное специально для Dodo Brands

## О проекте

Этот проект представляет собой одностраничное резюме, которое не просто рассказывает о навыках и опыте, но и демонстрирует их на практике. Каждый элемент интерфейса, каждая анимация и каждая деталь кода — это часть портфолио.

### Основные особенности

- **Современный стек**: React 19, TypeScript, Vite
- **Архитектура**: Feature-Sliced Design для масштабируемости
- **Анимации**: Framer Motion для плавных переходов
- **Темы**: поддержка светлой и темной темы с плавными переходами
- **Адаптивность**: полностью responsive дизайн для всех устройств
- **Производительность**: оптимизировано для быстрой загрузки и работы

## Структура проекта

Проект организован по методологии Feature-Sliced Design:

```
src/
├── app/                    # Инициализация приложения
│   ├── App.tsx           # Корневой компонент
│   ├── App.css           # Глобальные стили приложения
│   └── providers/        # Провайдеры контекста
│       └── themeStore.ts # Zustand store для управления темой
│
├── widgets/              # Крупные составные блоки интерфейса
│   ├── HeroSection/      # Главная секция с приветствием
│   ├── StackSection/     # Секция со стеком технологий
│   ├── ExperienceSection/# Секция с опытом работы
│   ├── ProjectsSection/ # Секция с проектами
│   ├── WhyDodoSection/  # Секция "Почему Dodo"
│   └── Footer/           # Футер с контактами
│
├── features/             # Функциональные возможности
│   └── ThemeToggle/      # Переключатель темы
│
├── shared/               # Переиспользуемые модули
│   ├── config/           # Конфигурация и константы
│   │   ├── constants.ts  # Глобальные константы
│   │   └── colors.ts     # Цветовая палитра
│   ├── lib/              # Утилиты и хелперы
│   │   └── hooks/        # Кастомные хуки
│   │       └── useIntersectionObserver.ts
│   ├── types/            # TypeScript типы
│   └── ui/               # UI компоненты
│       ├── Button/       # Кнопка
│       ├── Card/         # Карточка
│       └── ProgressBar/  # Прогресс-бар
│
└── index.tsx             # Точка входа
```

### Описание основных директорий

#### `app/`
Содержит инициализацию приложения и глобальные настройки. Здесь находится корневой компонент `App.tsx` и провайдеры для управления состоянием приложения.

#### `widgets/`
Крупные составные блоки интерфейса, которые объединяют несколько features и entities. Каждый виджет представляет собой отдельную секцию резюме:
- **HeroSection**: главная секция с именем, ролью и метриками
- **StackSection**: интерактивная секция со стеком технологий с фильтрацией
- **ExperienceSection**: timeline с опытом работы и образованием
- **ProjectsSection**: карточки проектов с достижениями
- **WhyDodoSection**: текстовая секция с мотивацией
- **Footer**: контакты и социальные сети

#### `features/`
Функциональные возможности, которые можно переиспользовать в разных частях приложения:
- **ThemeToggle**: переключатель темы с анимацией View Transitions API

#### `shared/`
Переиспользуемые модули, которые используются по всему приложению:
- **config**: константы, цвета, настройки
- **lib**: утилиты и кастомные хуки
- **types**: общие TypeScript типы
- **ui**: базовые UI компоненты

## 🛠 Технологический стек

- **React 19** — UI библиотека
- **TypeScript** — типизация
- **Vite** — сборщик и dev-сервер
- **Framer Motion** — анимации
- **Zustand** — управление состоянием
- **CSS Modules** — изоляция стилей

## 📦 Установка и запуск

### Требования

- Node.js 18+ 
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в директории `dist/`

### Превью продакшен-сборки

```bash
npm run preview
```

## Особенности реализации

### Управление темой

Тема приложения управляется через Zustand store с персистентностью в localStorage. Переключение темы использует View Transitions API для плавных переходов.

### Анимации

Все анимации реализованы через Framer Motion с использованием Intersection Observer для запуска при появлении элементов в viewport.

### Адаптивность

Проект полностью адаптивен и корректно отображается на всех устройствах:
- Мобильные устройства (< 768px)
- Планшеты (768px - 1024px)
- Десктоп (> 1024px)

## Структура данных

### Персональная информация

Хранится в `shared/config/constants.ts`:
- Имя, роль, локация
- Биография и качества
- Контакты

### Стек технологий

Определен в `widgets/StackSection/model/stackData.ts` с категориями:
- **expert**: экспертный уровень
- **advanced**: продвинутый уровень
- **familiar**: знаком

### Опыт работы

Хранится в `widgets/ExperienceSection/model/experienceData.ts` с информацией о:
- Компании и роли
- Периоде работы
- Достижениях
- Используемых технологиях

### Проекты

Определены в `widgets/ProjectsSection/model/projectsData.ts` с:
- Описанием проекта
- Достижениями
- Метриками
- Стеком технологий

## Настройка

### Изменение персональной информации

Отредактируйте файл `src/shared/config/constants.ts`:

```typescript
export const PERSONAL_INFO = {
  name: 'Ваше имя',
  role: 'Ваша роль',
  // ...
};
```

### Изменение цветовой схемы

Цвета определены в `src/shared/config/colors.ts`. Можно изменить палитру Dodo или добавить свои цвета.

### Добавление новых секций

1. Создайте новую директорию в `widgets/`
2. Создайте компонент секции
3. Импортируйте и добавьте в `App.tsx`
